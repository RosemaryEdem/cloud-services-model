# Cloud Service Providers and Models Assignment

## 1. Cloud Service Models (Real-Life Examples)

### IaaS (Infrastructure as a Service)
IaaS provides virtual computing resources such as servers, storage and networking over the internet.

- **Example:** **Amazon EC2** – I use this when I need full control over a virtual machine and operating system.
- **Other Examples:** Google Compute Engine, Microsoft Azure Virtual Machines.



### PaaS (Platform as a Service)
PaaS provides a platform for developing, testing and deploying applications without managing the underlying infrastructure.

- **Example:** **Render** (used in this project) – It automatically handles deployment, scaling and hosting.
- **Other Examples:** AWS Elastic Beanstalk, Google App Engine.



### SaaS (Software as a Service)
SaaS provides ready-to-use applications delivered over the internet.

- **Examples:** Gmail, WhatsApp, GitHub, Zoom.



## 2. Cloud Providers Research

### Amazon Web Services (AWS)
Amazon Web Services is one of the leading cloud computing providers.

**Core Services:**
- EC2 (Compute) – Virtual servers for running applications  
- S3 (Storage) – Object storage for files and backups  
- RDS (Databases) – Managed relational database service  



### Microsoft Azure
Microsoft Azure is a widely used cloud platform, especially in enterprise environments.

**Core Services:**
- Virtual Machines – Scalable compute resources  
- Blob Storage – Object storage for unstructured data  
- Azure SQL Database – Managed database service
