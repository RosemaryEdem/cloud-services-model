# Cloud Computing Models Presentation


## What is IaaS?
- Infrastructure as a Service
- Provides virtual servers, storage and networking
- Example: Amazon EC2


## What is PaaS?
- Platform as a Service
- Provides a platform to build and deploy applications
- **Service Used:** Render
- Benefit: Managed environment, no need to manage infrastructure


## What is SaaS?
- Software as a Service
- Ready-to-use applications over the internet
- Examples: GitHub, WhatsApp, Gmail
